# 政策跟踪器系统设计文档

## 一、系统概述

### 1.1 产品定位
面向企业的智能政策跟踪与申报管理系统，帮助企业：
- 自动发现可申请的政府政策
- 跟踪政策申报窗口期
- 管理申报材料清单
- 提醒关键时间节点

### 1.2 核心价值
- **省时**：自动匹配企业可申报政策，无需人工搜索
- **省心**：智能提醒申报窗口期，避免错过机会
- **省力**：材料清单自动生成，申报流程清晰

## 二、核心数据模型

### 2.1 政策（Policy）

```typescript
interface Policy {
  id: string;
  name: string;                    // 政策名称
  code: string;                    // 政策编号
  category: PolicyCategory;        // 政策类别
  level: PolicyLevel;              // 政策级别
  department: string;              // 发布部门
  
  // 时间信息
  publishDate: Date;               // 发布日期
  applyStartDate: Date;            // 申报开始日期
  applyEndDate: Date;              // 申报截止日期
  validUntil: Date;                // 政策有效期
  
  // 内容信息
  description: string;             // 政策简介
  eligibility: string[];           // 申报条件
  benefits: Benefit[];             // 政策奖励
  materials: Material[];           // 所需材料清单
  process: ProcessStep[];          // 申报流程
  
  // 状态
  status: PolicyStatus;            // 政策状态
  matchScore?: number;             // 与企业匹配度
}

enum PolicyCategory {
  HIGH_TECH = '国家高新技术企业',
  SPECIALIZED = '专精特新',
  RESEARCH = '研发补贴',
  TALENT = '人才政策',
  FINANCE = '融资支持',
  TAX = '税收优惠',
  INTELLECTUAL = '知识产权',
  OTHERS = '其他'
}

enum PolicyLevel {
  NATIONAL = '国家级',
  PROVINCIAL = '省级',
  MUNICIPAL = '市级',
  DISTRICT = '区级'
}

enum PolicyStatus {
  UPCOMING = '即将开始',
  OPEN = '申报中',
  CLOSING = '即将截止',
  CLOSED = '已截止',
  EXPIRED = '已过期'
}
```

### 2.3 申报材料（Material）

```typescript
interface Material {
  id: string;
  name: string;                    // 材料名称
  category: MaterialCategory;      // 材料类别

  template?: string;               // 模板文件链接
  format: string[];                // 支持格式
  maxSize: number;                 // 最大文件大小(MB)
  
  // 智能生成
  aiGeneratable: boolean;          // 是否支持AI生成
  aiPrompt?: string;               // AI生成提示词
}

enum MaterialCategory {
  BASIC = '基础材料',              // 营业执照、法人身份证等
  FINANCIAL = '财务材料',          // 财务报表、审计报告
  TECHNICAL = '技术材料',          // 专利证书、技术说明
  PROJECT = '项目材料',            // 项目计划书、可行性报告
  OTHER = '其他材料'
}
```

```typescript
interface ApplicationTask {
  id: string;

  SUBMITTED = '已提交',
  APPROVED = '已通过',
```

## 四、核心功能清单

### 4.1 政策发现

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 政策抓取 | 自动抓取各级政府政策信息 | P0 |
| 智能解析 | NLP解析政策内容，提取关键信息 | P0 |
| 企业匹配 | 根据企业画像匹配可申报政策 | P0 |
| 匹配评分 | 计算政策匹配度评分 | P1 |
| 推荐排序 | 按匹配度和奖励金额排序 | P1 |

### 4.2 窗口期跟踪

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 时间识别 | 自动识别申报开始/截止时间 | P0 |
| 状态计算 | 实时计算政策状态（即将开始/申报中/即将截止） | P0 |
| 日历视图 | 展示申报时间线 | P1 |
| 倒计时提醒 | 截止日倒计时 | P1 |
| 历史记录 | 已截止政策归档 | P2 |

### 4.3 材料管理

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 清单生成 | 根据政策生成材料清单 | P0 |
| 材料上传 | 支持多种格式上传 | P0 |
| 模板下载 | 提供标准材料模板 | P1 |
| AI生成 | 智能生成申报材料 | P1 |
| 材料审核 | 自动检查材料完整性 | P2 |
| 版本管理 | 材料版本控制 | P2 |

### 4.4 提醒通知

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 多渠道通知 | 微信/短信/邮件/APP推送 | P0 |
| 自定义提醒 | 支持自定义提醒时间 | P1 |
| 智能提醒 | 根据进度自动提醒 | P1 |
| 提醒优先级 | 紧急事项优先提醒 | P2 |
